# AP8.Sudoku

Lab assignment for the Unit 8 AP CS-A class, with focus on recursion.
This assignment is a Sudoku puzzle solver. It is using a graphical user interface (GUI) library to render the puzzle and to show the progression of the algorithm.  

The content needed for this assignment:
* [ ] [SudokuSolver](https://florinteo.github.io/EduLabs/ap8.Sudoku/spec/SudokuSolver.pptx): complete specification of the project.
* [ ] [drawing-lib](https://florinteo.github.io/EduLabs/ap8.Sudoku/lib/drawing-lib-2.2-shaded.jar): Java API library supporting graphic operations such as rendering a puzzle and showing the algorithm progress. It is available in the assignment package, in the *lib* folder. Instructions on how to configure this library in your environment can be found [here](https://github.com/LWHSStave/Setup/blob/main/RefJar/README.md).
* [ ] [SudokuBoard](https://florinteo.github.io/EduCode/DrawingLib/edu/ftdev/Sudoku/SudokuBoard.html) class: detailed documentation of the class contained in the `drawing-lib` library, the only one needed for this project.
* [ ] [SudokuRunner.java](https://florinteo.github.io/EduLabs/ap8.Sudoku/src/SudokuRunner.java) startup code: showcases the usage of the SudokuBoard GUI API and contains the `main()` entry point. Also available in the assignment package, in the *src* folder.
