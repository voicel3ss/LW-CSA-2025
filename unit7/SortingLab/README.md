# AP7.SortingLab

Lab assignment for the Unit 7 AP CS-A class, with focus on sorting and big-Oh commplexity.

This assignment is about implementing the insertion and selection sorting algorithms, and display their execution time (latency) in a GUI window. Based on that, analyze and discuss the big-Oh complexity of each of these algorithms.

The content needed for this assignment:
* [ ] [Insertion_Sort.pptx](https://florinteo.github.io/EduLabs/ap7.SortingLab/spec/Insertion_Sort.pptx): Detailed description of the insertion sort algorithm.
* [ ] [Selection_Sort.pptx](https://florinteo.github.io/EduLabs/ap7.SortingLab/spec/Selection_Sort.pptx): Detailed description of the selection sort algorithm.
* [ ] [graph.jar](https://florinteo.github.io/EduLabs/ap7.SortingLab/lib/graph.jar): Small specialized GUI library to assist with graphing out datasets.
