# AP7.Elevens

Lab assignment for the Unit 7 AP CS-A class, with focus on searching, sorting, shuffling. It goes beyond AP-CS curriculum, introducing abstract classes and interfaces.  

This assignment is about implementing the Elevens cards game. It is organized in eleven activities of increased complexity, 
culminating with a GUI to give a visual representation of the game.
